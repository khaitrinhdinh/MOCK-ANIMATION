How it's built

  • Real create-song flow — the same wizard as the LIFF app: seed → image → mood → vocal → text → confirm, with the step dots, chat bubbles and inline options.
  • Fully mocked — sample seeds & moods, image kept as a local preview only. No upload, no Mureka call; it just runs on-hand data to test the UI.
  • Drawn processing screen — the loader here is pure SVG/CSS: a glowing seed, a growing sprout and a dripping dew drop, synced to the three "Analyzing / Lyrics / Composing" steps.
  • Ends on the result — progress fills, then the completed screen offers Play / Create Another.

This is a front-end mock of the create page. In the real app the same setStep() machine drives it and the backend fills progress; here a timer stands in.
