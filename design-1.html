How it's built

  • Falling particles & rain — each one is an SVG shape on a looping @keyframes, offset with animation-delay so it feels random.
  • Roots, stems & branches growing — the stroke-dashoffset trick: draw the path, then reveal the stroke from zero.
  • Leaves, foliage & petals popping in — transform: scale(0→1) with transform-origin at the base, plus a springy easing.
  • Fireflies & night glow — drifting translate loops with a CSS drop-shadow glow; the whole screen re-themes via CSS variables.
  • Music drop & ripples — translateY + scale, final state held with fill-mode: forwards.
… total weight of this entire page — all four designs, HTML + CSS + JS + artwork + embedded font, before gzip. The Rive/Lottie runtime alone is ~100 KB.

This demo simulates progress over 13 seconds. In the real app, swap the timer for the actual status returned by the music-generation API — each phase is just a CSS class (ph1/ph2/ph3/done) on the stage; CSS handles the rest.
